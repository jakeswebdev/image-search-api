Image Search API

This little API returns a json file with different image search results based on a search term.

To do a search, make a GET request the following way:

https://OUR_URL/search/YOUR_SEARCH_TERM

  • Example: https://imgnow.glitch.me/search/puma

You can also look for the most recent search terms at:

https://OUR_URL/recent

  • Example: https://imgnow.glitch.me/recent

    • If you want to use an offset so that the search returns results after a certain index, you can pass "&start=" as a URL parameter.

    • Example: https://imgnow.glitch.me/search/puma&start=2
